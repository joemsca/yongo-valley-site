# Yongo Valley Ventures

Public marketing site for [Yongo Valley Ventures](https://yongovalley.com) — owner-operator holding company.

Static HTML, hosted on Vercel (`joemscas-projects/yongo-valley-site`).

## Local preview

Open `index.html` in a browser, or:

```bash
npx serve .
```

## Deploy

Pushes to `main` deploy to production on Vercel. Preview URLs are issued on every push/PR.
